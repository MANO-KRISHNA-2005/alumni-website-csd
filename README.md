# alumni-website
Website for Alumni Meet 2026

**TEAMMATE GUIDE
**

  Step 1: Clone Repository
  
    git clone https://github.com/YOURUSERNAME/alumni-website.git
    
    cd alumni-website
  
  Step 2: Create Feature Branch
  
    # Always create new branch for each feature
    
    git checkout -b feature/your-name-feature
    
    # Examples:
    
    git checkout -b feature/priya-registration
    
    git checkout -b feature/amit-gallery
    
    git checkout -b feature/raj-homepage
  
  Step 3: Make Changes
  
    Edit files in src/ folder
    
    Add new features/fixes
    
    Test your changes
  
  Step 4: Save & Push Changes
  
    # Check what you changed
    
    git status
    # Add all changes
    git add .
    # Commit with description
    git commit -m "Added registration form with validation"
    # Push to your branch
    git push origin feature/your-name-feature
